Getting Started with Tailwind CSS

Kick-start your styling with utility classes and a zero-config CDN setup. Learn how to build responsive layouts quickly and efficiently.

Why Tailwind?

Tailwind CSS is a utility-first framework that gives you the power to build completely custom designs without leaving your HTML. It eliminates the need for writing custom CSS by providing hundreds of utility classes for layout, spacing, typography, and more.

  • Design directly in your markup
  • Highly responsive and mobile-first
  • Minimal configuration needed to get started

Setting Up via CDN

You can start using Tailwind instantly by including it through a CDN. Add the following script inside your HTML’s <head> tag: 

<script src="https://cdn.tailwindcss.com"></script>

This approach is perfect for learning, quick prototypes, or small projects.

Example Button

Here’s how to create a styled button using Tailwind’s utility classes: 

<button class="bg-indigo-600 hover:bg-indigo-700 text-white font-semibold py-2 px-4 rounded">
  Click Me
</button>

What's Next?

Once you're familiar with Tailwind’s utility classes, you can move on to advanced topics like custom themes, dark mode support, or integrating with frameworks like React, Vue, or Svelte.
